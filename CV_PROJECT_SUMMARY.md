# VisiCalc - CV Project Summary

## Quick Project Description for CV/Resume

**VisiCalc - C++ Terminal Spreadsheet Application**

A feature-rich spreadsheet application demonstrating advanced C++ programming, object-oriented design, and algorithm implementation.

### Key Technical Achievements:
- **Object-Oriented Architecture**: Implemented polymorphic cell hierarchy with abstract base classes
- **Formula Parser**: Built recursive descent parser supporting arithmetic operations and statistical functions (SUM, AVERAGE, STDDEV, MAX, MIN)
- **Memory Management**: Utilized modern C++ smart pointers (std::unique_ptr) for automatic resource management
- **Template Programming**: Designed generic Grid<T> template class for type-safe data storage
- **File I/O**: Implemented CSV import/export functionality
- **Circular Reference Detection**: Developed algorithm to prevent infinite loops in formula calculations

### Technologies Used:
- C++17 (STL, Smart Pointers, Templates)
- ANSI Terminal Control
- Object-Oriented Design Patterns
- Recursive Descent Parsing
